# ComparativyStudyForecastLEO
This repo has the datasets used as well as some of the codes presentend in Based Traffic Forecasting for Beam Hopping LEO
Satellite Networks paper that is in press.

To train DLinear, we used the source code given in https://github.com/vivva/DLinear

To train Informer, we used the source code given in https://github.com/zhouhaoyi/Informer2020

To train chronos, we used chronos-bolt-tiny model given in https://github.com/amazon-science/chronos-forecasting

For FARIMA we used the arfima package given in https://cran.r-project.org/web/packages/arfima/index.html
